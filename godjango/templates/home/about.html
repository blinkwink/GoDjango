About GoDjango

GoDjango is created by Buddy Lindsey. Each episode is focused on something that can help django developers do django development better. Most videos will be short, sweet, and to the point about the specific topic. It is expected that most developers have some experience with Django, but anyone will usually be able to benefit some how.

Contact

All feedback is welcome be it comment, criticism, help, or suggestions feel free to hit me with it. The easiest way to contact me is via the feedback page. However, I am available on twitter as well at, @BuddyLindsey.

GoDjango Site

The GoDjango site itself is open source so anyone can feel free to peruse the code, or submit pull requests to help grow the site itself. Please have a look at the GoDjango github repository to see more about it. Also please feel free to leave pull requests.
